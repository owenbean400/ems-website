# Delete Organization

## DELETE `/v1/ems/organization/{org_id}`

Deletes the organization. This is idempotent whether the record exist or not as long as the record exist is the owner making the call.

### Request

Headers

```
"Authorization": "Bearer <Token>"
```

| Path Parameters | |
| --------------- | - |
| org_id | The organization ID |

### Response `204`

The organization has been deleted.

### Response `401`

Authorization issue.

#### Body
```
{
    "Error": "Unauthorized",
    "message": [string] Authorization issue
}
```

#### Messages

- You are not authorized to access this resource
> The token from Firebase is invalid
- Authorization header is missing
> There is no "Authorization" in the header request.
- Invalid authorization header format
> The "Authorization" header does not have "Bearer Token" format.

### Response `404`

The organization does not exist.

### Response `500`

Internal error with the server.

#### Body
```
{
    "error": "Internal Server Error",
    "message": "Internal Server Error"
}
```
